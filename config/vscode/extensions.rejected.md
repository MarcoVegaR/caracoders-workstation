# VS Code extensions rejected in v1

| Extension/category | Decision | Reason |
|---|---:|---|
| SQLTools | No instalar | La inspección SQL v1 se hace con pgAdmin web, `psql`, contenedor y MCP PostgreSQL. |
| PostgreSQL extension | No instalar | Redundante para la estrategia oficial y aumenta superficie de configuración. |
| Thunder Client | No instalar por defecto | HTTPie, Playwright y herramientas de proyecto cubren v1. |
| REST Client | No instalar por defecto | Evita fragmentar el flujo de pruebas. |
| Extensiones IA duplicadas | No instalar | OpenCode es el agente principal. |
