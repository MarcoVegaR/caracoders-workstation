# VS Code

VS Code is a support tool for reading, diagnostics, Git review, Dev Containers and editing. OpenCode, CLI, Docker, Docker Compose, Dev Containers and MCP are the automation layer.

## VS Code vs MCP vs CLI matrix

| Tool | Classification | Reason |
|---|---:|---|
| GitLens | instalar | Visual Git review |
| Docker extension | instalar | Container visibility |
| Dev Containers | instalar | Required runtime workflow |
| Error Lens | instalar | Diagnostics |
| PHP Intelephense | instalar | PHP navigation |
| Laravel Artisan | instalar | Laravel convenience |
| ESLint | instalar | JS diagnostics |
| Prettier | instalar | Formatting |
| Playwright Test | instalar | Test runner support |
| SQLTools | no instalar | Redundant with pgAdmin, psql and MCP |
| PostgreSQL extension | no instalar | Redundant in v1 |
| Thunder Client | futuro | Not baseline |
| REST Client | futuro | Not baseline |
